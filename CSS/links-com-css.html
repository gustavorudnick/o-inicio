Interações com Links

Link com <active>

O estilo do link muda quando precionado

Link com <visited>

O estilo do link muda quando visitado

Link com <hover>

O estilo do link muda quando o mouse passa por ele

Link com <cursor: "tipo">

O cursor do mouse muda quando passa pelo link. Se não especificado, fica o padrão
